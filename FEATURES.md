# Features
このプロジェクトに追加する予定、または追加したい機能をまとめたリストです

## サーバー側
- 使用率をモニタリングする際小数点以下が0のときにスタートして、複数からの接続があっても一回の取得で全体に送信する機能
- 複数台のサーバーを管理できるように、エントリー用のサーバーを静的ファイルホスティングかつAPIのエントリーポイントにして、管理したいサーバー側にはエントリー用サーバーのためのREST APIサーバーを設置して、エントリーサーバーを介して情報にアクセスできるようにする
- エントリーサーバーから起動してない登録済みのサーバーにマジックパケットを送信して、遠隔で起動できるようにする

## クライアント側
- TypeScriptのコーディングにはAIを使用しているため、保守性の向上かつデバッグ作業をする
